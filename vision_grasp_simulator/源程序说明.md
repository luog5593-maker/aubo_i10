# 源程序说明

本项目采用分层结构组织源程序。`app`目录负责桌面界面和用户交互；`core`目录负责视觉检测、坐标转换、运动学、轨迹规划、抓取流程、安全判断以及 AUBO i10/PGC_300_60 模型加载；`utils`目录负责图像、文件和日志工具；`config`目录保存运行参数；`assets`目录用于存放可选的 AUBO i10 URDF、DAE/STL mesh 与 PGC_300_60 STL mesh 资源。

源程序原创部分主要体现在 GUI 界面、模拟检测、坐标转换、机械臂运动学、MoveJ/MoveL 仿真、自动抓取流程、安全判断、日志导出、AUBO i10 URDF 解析、mesh 路径映射、真实外观模型/程序化模型双模式显示、电爪开合仿真和 AUBO i10 与 PGC_300_60 联合运动仿真。

本系统不连接真实机械臂，不依赖 ROS、Gazebo、RViz、MoveIt 或 AUBO SDK。模型文件缺失时，程序自动回退到程序化几何体显示，保证普通 Windows 电脑上可以完成视觉识别与机械臂抓取仿真演示。
