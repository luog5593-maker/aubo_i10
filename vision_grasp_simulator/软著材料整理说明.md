# 软著材料整理说明

建议将以下内容整理为软件著作权材料：
1. 软件名称统一使用《基于YOLO的机械臂视觉识别与抓取仿真系统 V1.0》。
2. 源程序鉴别材料可优先选取`app/main_window.py`、`app/widgets/robot_view_panel.py`、`core/robot_model_loader.py`、`core/vision_detector.py`、`core/coordinate_transform.py`、`core/robot_kinematics.py`、`core/trajectory_planner.py`、`core/grasp_simulator.py`、`core/safety_checker.py`等文件。
3. 说明书截图建议包含主界面、AUBO i10 + PGC_300_60 三维组合模型、示例图像检测结果、自动抓取过程、电爪打开/闭合状态、日志导出结果。
4. 文档表述应说明本系统支持调用YOLO模型，不表述为原创YOLO算法；当前版本为纯软件仿真版本，不表述为真实机械臂控制系统。
5. 文档可说明当前版本支持 AUBO i10 URDF 参数解析、DAE/STL 外观模型加载、PGC_300_60 电爪模型加载、电爪开合仿真以及机械臂与电爪联合运动仿真。
