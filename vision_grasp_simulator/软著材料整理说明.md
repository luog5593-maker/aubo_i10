# 软著材料整理说明

建议将以下内容整理为软件著作权材料：
1. 软件名称统一使用《基于YOLO的机械臂视觉识别与抓取仿真系统 V1.0》。
2. 源程序鉴别材料可优先选取`app/main_window.py`、`core/vision_detector.py`、`core/coordinate_transform.py`、`core/robot_kinematics.py`、`core/trajectory_planner.py`、`core/grasp_simulator.py`、`core/safety_checker.py`等文件。
3. 说明书截图建议包含主界面、示例图像检测结果、三维机械臂模型、自动抓取过程、日志导出结果。
4. 文档表述应说明本系统支持调用YOLO模型，不表述为原创YOLO算法；当前版本为纯软件仿真版本，不表述为真实机械臂控制系统。
