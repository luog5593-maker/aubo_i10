# aubo_i10

本项目用于开发“基于机器视觉的机械臂目标识别与抓取辅助系统”。

## 项目目标

1. 建立 Python 软件项目结构；
2. 实现图像导入、图像预处理、目标检测结果显示；
3. 后续扩展机械臂抓取点辅助计算、实验数据管理和报告导出；
4. 项目可用于课程展示、科研辅助和软件著作权申请。

## 初步功能模块

1. 图像导入模块；
2. 图像预处理模块；
3. 目标检测结果显示模块；
4. 机械臂抓取点辅助分析模块；
5. 检测报告导出模块；
6. 用户操作日志模块。

## 开发说明

当前项目处于初始阶段，后续将使用 Python 进行开发，优先完成离线软件原型，不直接连接真实机械臂设备。
